# Cyfer Streaming Releases

This public repository is the release feed for Cyfer Streaming installers and auto-update metadata.

The application source code is not published here. Releases contain only packaged installer artifacts generated by `electron-builder`, including the Windows installer and `latest.yml` used by the in-app updater.

## Downloads

Use the Releases page to download the latest public build:

https://github.com/FortunasXP/Cyfer-Streaming-Releases/releases

## Auto Updates

Installed Cyfer builds check this repository for update metadata. If an update is available, the app can download it and install it on restart.
